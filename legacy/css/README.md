# css — الستايل

> **داخل مستودع `sl-dubbing-frontend-main`**

---

## الملفات

### style.css
الستايل الرئيسي للمشروع كله:
- **الألوان:** متغيرات CSS (`--primary`, `--bg`, `--text`, ...)
- **Dark mode:** `@media (prefers-color-scheme: dark)` + `.dark-mode` class
- **Layout:** CSS Grid + Flexbox
- **Responsive:** breakpoints لـ mobile/tablet/desktop
- **المكونات:** buttons, inputs, cards, progress bars, modals, toasts
- **الصفحات:** أقسام خاصة بـ dubbing, tts, history, pricing

### logo.css
ستايل Logo المتحرك:
- animation لـ SVG Logo
- أحجام مختلفة (navbar, hero, loading)
