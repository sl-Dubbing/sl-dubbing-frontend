# demo — أمثلة الدبلجة

> **داخل مستودع `sl-dubbing-frontend-main`**
> أمثلة تُعرض في الصفحة الرئيسية (`index.html`) لإظهار جودة الدبلجة.

---

## الملفات

| الملف | الوصف |
|-------|-------|
| `EN.mp4` | مثال دبلجة: فيديو بالإنجليزية مدبلج للعربية |
| `IT.mp4` | مثال دبلجة: فيديو بالإيطالية مدبلج للعربية |
| `portrait.png` | صورة للعرض في قسم الأمثلة |
| `script-en.txt` | نص السكريبت الإنجليزي للمثال الأول |

هذه الملفات **static assets** — تُقدَّم مباشرة ولا علاقة لها بكود التطبيق.

**SEO / Performance:** keep `portrait.png` under ~150 KB (WebP or compressed PNG). Demo videos (`EN.mp4`, `IT.mp4`) should be H.264, short clips, and under ~5 MB each for fast homepage load.
